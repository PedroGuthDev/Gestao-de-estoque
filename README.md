Projeto de sistema de gerenciamento de estoque criado com o objetivo de treinar rotas, manipulação de estado e persistência de dados.

## 🚀 Sobre

Este projeto é uma aplicação web (SPA) para gerenciar o inventário de produtos, com foco na prática de operações CRUD com React Router, manipulação de estado e LocalStorage.

## 💡 Funcionalidades

- ➕ Registrar novos produtos ao estoque
- 📝 Editar informações (preço, quantidade, categoria, etc.)
- 📉 Dashboard dinâmico com ites recentes e produtos com estoque baixo
- 🔄 Navegação fluida entre páginas (React Router)
- 🧠 Persistência de dados usando LocalStorage

## 🧰 Tecnologias Utilizadas

- React (Vite) - Estrutura da aplicação single-page
- TypeScript - Tipagem estática
- React Router - Gerenciamento das rotas e loaders
- Bootstrap - Estilização visual e responsividade
- LocalStorage - Armazenamento local (Banco de dados simulado)

## 📌 Como Usar

Clone o repositório:

```bash
git clone https://github.com/PedroGuthDev/Gestao-de-estoque.git
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Abra o terminal/link gerado (padrão: localhost:8080) no seu navegador.
